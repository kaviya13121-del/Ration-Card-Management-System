# 🏛️ Ration Card Management System

## 📖 Project Overview

The Ration Card Management System is a web-based application developed using PHP and MySQL. It simplifies the process of managing ration card records by providing a digital platform for storing, updating, and managing beneficiary information. The system replaces manual paperwork with an efficient, secure, and user-friendly solution.

---

## 🎯 Objectives

- Digitize ration card management.
- Reduce manual paperwork.
- Improve record accuracy.
- Provide secure login for administrators.
- Simplify data management and retrieval.

---

## ✨ Features

- 🔐 User Login Authentication
- 📋 Dashboard for Admin
- 🗂️ Ration Card Record Management
- ➕ Add New Records
- ✏️ Update Existing Records
- ❌ Delete Records
- 🔍 Search Records
- 🗄️ MySQL Database Integration

## ⚙️ Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into:

```
C:\xampp\htdocs\
```

4. Import the database into phpMyAdmin.
5. Open your browser and visit:

```
http://localhost/rationproject/
```
